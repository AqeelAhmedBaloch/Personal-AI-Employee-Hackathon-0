## 🎯 **PLATINUM TIER FEATURES**

### **What is Platinum Tier?**

Platinum Tier transforms your AI Employee from a local tool to a **24/7 cloud-based autonomous worker** with:

1. **Cloud Deployment** - Always-on watchers on Oracle/AWS VM
2. **Work-Zone Specialization** - Cloud drafts, Local approves
3. **Vault Sync** - Git-based synchronization
4. **Security** - Secrets never sync to cloud
5. **Claim-by-Move** - Prevents double-work between agents
6. **A2A Communication** - Direct agent messaging

---

## 📊 **COMPARISON: GOLD vs PLATINUM**

| Feature | Gold Tier | Platinum Tier |
|---------|-----------|---------------|
| **Deployment** | Local only | Cloud + Local |
| **Availability** | When running | 24/7 always-on |
| **Agents** | Single agent | Cloud + Local agents |
| **Sync** | Not needed | Git-based vault sync |
| **Security** | Local secrets | Cloud/local separation |
| **Task Ownership** | Single agent | Claim-by-move rule |
| **Communication** | File-based | A2A messaging |

| **Time** | 40+ hours | 60+ hours |
